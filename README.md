# Kalyan Elevators Private Limited

Luxury B2B single-page website for **Kalyan Elevators Private Limited**, Thrissur, Kerala.

> Elevating Kerala's Architectural Standards.

## Tech stack

- **Vite** + **React 18** (functional components, hooks)
- **Tailwind CSS** (Obsidian + Champagne Gold theme)
- **GSAP** + ScrollTrigger (reveal + cabin simulator animations)
- **Lucide React** icons
- **Web Audio API** synthesized arrival chime

## Getting started

```bash
npm install
npm run dev      # start dev server
npm run build    # production build
npm run preview  # preview the build
```

## Features

- Immersive hero with animated elevator-shaft background
- Live stats bar, About section, Safety/ARD USPs
- Interactive Cabin Simulator (finish, lighting, floor stops with sound)
- 6-category elevator catalog with WhatsApp inquiry per product
- Services & AMC, testimonial slider
- 4-step Quote Builder modal that hands off a pre-composed WhatsApp message
- Sticky glassmorphic action dock (Call / WhatsApp / Quote)
- Dark-themed embedded Thrissur map and contact section

All CTAs route to the Quote modal, WhatsApp (`+91 79949 99926`) or a direct call.
No e-commerce / checkout.
